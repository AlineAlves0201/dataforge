# 13 - Preparar PostgreSQL

## Objetivo

Preparar um banco relacional para persistir dados.

## Entrega

PostgreSQL disponivel localmente, via instalacao direta ou Docker.

## Checklist

- Escolher instalacao direta ou Docker.
- Criar banco `dataforge`.
- Criar usuario ou usar usuario local.
- Testar conexao.

## Criterio de aceite

E possivel conectar no banco `dataforge` e executar `select 1`.
