# 02 - Configurar identidade do Git

## Objetivo

Configurar nome e e-mail para que os commits fiquem associados corretamente.

## Entrega

Git global configurado com `user.name` e `user.email`.

## Checklist

- Rodar `git config --global user.name "Seu Nome"`.
- Rodar `git config --global user.email "seu-email-do-github@email.com"`.
- Conferir com `git config --global --list`.

## Criterio de aceite

`user.name` e `user.email` aparecem corretamente na listagem de configuracao.
