# 11 - Replicar pipeline para segunda API

## Objetivo

Reaplicar a logica de consumo e salvamento usando outra fonte.

## Entrega

Segundo pipeline de API funcionando.

## Checklist

- Criar modulo ou funcao para a nova API.
- Fazer chamada HTTP.
- Tratar erro.
- Salvar JSON raw.
- Versionar o codigo.

## Criterio de aceite

O pipeline da segunda API roda sem depender do codigo especifico da PokeAPI.
