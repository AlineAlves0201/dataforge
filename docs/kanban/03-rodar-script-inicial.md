# 03 - Rodar script inicial

## Objetivo

Executar o primeiro script Python do projeto.

## Entrega

Arquivo `data/raw/user_profile.json` gerado pelo `src/main.py`.

## Checklist

- Ativar o `.venv`.
- Rodar `python src\\main.py`.
- Informar nome e idade.
- Abrir o JSON gerado em `data/raw/user_profile.json`.

## Criterio de aceite

O JSON contem `name`, `age` e `estimated_days_lived`.
