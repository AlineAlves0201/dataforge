# 08 - Salvar resposta da API em JSON

## Objetivo

Persistir a resposta bruta da API na camada raw.

## Entrega

Arquivo JSON salvo em `data/raw/`.

## Checklist

- Criar funcao para salvar JSON.
- Salvar a resposta completa da PokeAPI.
- Nomear o arquivo de forma clara, como `pokemon_pikachu.json`.
- Garantir encoding UTF-8 e indentacao.

## Criterio de aceite

O arquivo gerado pode ser aberto e contem a resposta completa da API.
