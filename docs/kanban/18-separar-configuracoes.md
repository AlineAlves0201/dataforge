# 18 - Separar configuracoes

## Objetivo

Remover configuracoes sensiveis ou variaveis de dentro do codigo.

## Entrega

Configuracoes lidas de ambiente ou arquivo local ignorado pelo Git.

## Checklist

- Criar exemplo `.env.example`.
- Usar `.env` local ignorado pelo Git.
- Ler parametros no codigo.
- Documentar variaveis necessarias.

## Criterio de aceite

O codigo nao contem credenciais nem configuracoes fixas de ambiente local.
