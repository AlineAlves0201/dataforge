# 15 - Inserir dados via Python

## Objetivo

Conectar Python ao PostgreSQL e inserir dados no banco.

## Entrega

Script Python gravando dados da API no PostgreSQL.

## Checklist

- Adicionar driver PostgreSQL ao `requirements.txt`.
- Criar funcao de conexao.
- Inserir payload raw.
- Confirmar insercao com SQL.

## Criterio de aceite

Dados consumidos pela API aparecem em uma tabela PostgreSQL.
