# 27 - Instalar PySpark

## Objetivo

Entrar na etapa de processamento distribuido com Spark.

## Entrega

PySpark instalado e validado.

## Checklist

- Adicionar `pyspark` ao `requirements.txt`.
- Instalar dependencias.
- Criar script minimo com `SparkSession`.
- Rodar uma leitura simples.

## Criterio de aceite

Um script PySpark executa localmente e cria uma `SparkSession`.
