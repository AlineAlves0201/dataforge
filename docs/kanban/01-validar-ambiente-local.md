# 01 - Validar ambiente local

## Objetivo

Confirmar que o computador esta pronto para executar a Sprint 1 do DataForge.

## Entrega

Ambiente validado com Python, Git, GitHub CLI, VS Code e virtualenv.

## Checklist

- Abrir um PowerShell novo.
- Entrar na pasta do projeto.
- Validar `python --version`.
- Validar `git --version`.
- Validar `gh --version`.
- Ativar o ambiente virtual com `.\\.venv\\Scripts\\Activate.ps1`.

## Criterio de aceite

Todos os comandos respondem com versao e o terminal mostra o ambiente `.venv` ativo.
