# 14 - Modelar tabelas raw e trusted

## Objetivo

Criar uma separacao simples entre dado bruto e dado tratado.

## Entrega

Tabelas SQL para armazenar dados raw e trusted.

## Checklist

- Definir tabela raw com payload JSON.
- Definir tabela trusted com campos normalizados.
- Criar script SQL em `docs/` ou pasta apropriada.
- Executar o script no banco.

## Criterio de aceite

As tabelas existem e refletem as camadas raw e trusted.
