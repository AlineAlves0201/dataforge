# 07 - Consumir PokeAPI

## Objetivo

Fazer a primeira chamada real para uma API publica.

## Entrega

Script que busca dados de um Pokemon na PokeAPI.

## Checklist

- Criar modulo para chamadas de API.
- Fazer `GET` para `https://pokeapi.co/api/v2/pokemon/pikachu`.
- Exibir status code.
- Exibir campos principais da resposta.

## Criterio de aceite

O script retorna dados reais da API e mostra status code `200`.
