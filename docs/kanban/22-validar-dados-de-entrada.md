# 22 - Validar dados de entrada

## Objetivo

Garantir que dados minimos existem antes de salvar ou transformar.

## Entrega

Validacoes basicas nos payloads consumidos.

## Checklist

- Definir campos obrigatorios.
- Validar tipos principais.
- Tratar campos ausentes.
- Logar payload invalido sem derrubar tudo sem contexto.

## Criterio de aceite

Payload invalido gera erro claro e nao passa para trusted.
