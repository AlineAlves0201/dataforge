# 25 - Criar docker compose

## Objetivo

Orquestrar aplicacao e PostgreSQL localmente.

## Entrega

`docker-compose.yml` com servicos do projeto.

## Checklist

- Definir servico `app`.
- Definir servico `postgres`.
- Configurar variaveis de ambiente.
- Configurar volume do banco.
- Testar subida dos containers.

## Criterio de aceite

`docker compose up` sobe app e banco sem erro critico.
