# 04 - Versionar primeira entrega

## Objetivo

Criar o primeiro commit local do DataForge.

## Entrega

Commit inicial com estrutura, README, script Python e documentacao.

## Checklist

- Rodar `git status`.
- Rodar `git add .`.
- Rodar `git commit -m "chore: initial dataforge setup"`.
- Conferir `git log --oneline`.

## Criterio de aceite

O historico do Git mostra o commit inicial.
