# 20 - Adicionar testes basicos

## Objetivo

Comecar a validar comportamento do codigo automaticamente.

## Entrega

Primeiros testes com `pytest`.

## Checklist

- Adicionar `pytest` ao `requirements.txt`.
- Criar pasta `tests/`.
- Testar funcao pura, como montagem de payload ou caminho de arquivo.
- Rodar `python -m pytest`.

## Criterio de aceite

Os testes passam e podem ser executados por comando unico.
