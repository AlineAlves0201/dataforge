# 19 - Criar funcoes reutilizaveis

## Objetivo

Reduzir duplicacao e deixar o pipeline mais facil de evoluir.

## Entrega

Funcoes pequenas e reutilizaveis para etapas comuns.

## Checklist

- Criar funcao de requisicao HTTP.
- Criar funcao de salvamento JSON.
- Criar funcao de leitura de configuracao.
- Evitar funcoes muito grandes.

## Criterio de aceite

Adicionar uma nova API exige pouca duplicacao de codigo.
