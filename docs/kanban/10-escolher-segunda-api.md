# 10 - Escolher segunda API

## Objetivo

Escolher outra API para provar autonomia fora do exemplo guiado.

## Entrega

API escolhida e documentada.

## Checklist

- Pesquisar APIs publicas simples.
- Confirmar se a API exige chave.
- Documentar URL base.
- Documentar exemplo de endpoint.

## Criterio de aceite

A nota da tarefa informa API escolhida, endpoint inicial e motivo da escolha.
