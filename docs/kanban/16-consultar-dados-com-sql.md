# 16 - Consultar dados com SQL

## Objetivo

Praticar consultas SQL sobre os dados persistidos.

## Entrega

Arquivo com consultas SQL uteis para explorar o dataset.

## Checklist

- Criar consultas com `select`.
- Usar `where`.
- Usar `order by`.
- Usar agregacoes simples.
- Salvar consultas em `docs/`.

## Criterio de aceite

As consultas retornam resultados e ajudam a entender os dados coletados.
