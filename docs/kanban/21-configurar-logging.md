# 21 - Configurar logging

## Objetivo

Substituir prints soltos por logs mais profissionais.

## Entrega

Pipeline emitindo logs informativos.

## Checklist

- Configurar modulo `logging`.
- Registrar inicio e fim do pipeline.
- Registrar chamadas de API.
- Registrar arquivos ou registros gravados.

## Criterio de aceite

Ao rodar o pipeline, os logs mostram o caminho principal da execucao.
