# 06 - Instalar dependencias de API

## Objetivo

Preparar o projeto para consumir APIs HTTP com Python.

## Entrega

Dependencia `requests` adicionada ao projeto.

## Checklist

- Adicionar `requests` ao `requirements.txt`.
- Rodar `python -m pip install -r requirements.txt`.
- Validar importando `requests` em um terminal Python.
- Commitar a mudanca.

## Criterio de aceite

`requests` instala sem erro e fica registrado em `requirements.txt`.
