# 24 - Criar Dockerfile

## Objetivo

Empacotar a aplicacao Python em uma imagem Docker.

## Entrega

`Dockerfile` capaz de instalar dependencias e rodar o pipeline.

## Checklist

- Escolher imagem base Python.
- Copiar arquivos necessarios.
- Instalar `requirements.txt`.
- Definir comando padrao.

## Criterio de aceite

A imagem Docker builda sem erro.
